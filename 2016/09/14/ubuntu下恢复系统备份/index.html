ubuntu下恢复系统备份

发表于 2016-09-14   |   分类于 技术

前提是已经有一个备份好的备份压缩文件，只是需要恢复覆盖到新系统上。

  1. 制作一个ubuntu的启动盘，我用的UltraISO，使用方法在另一篇里会提到。

  2. 启动盘引导进入以后，点击try ubuntu，也就是进入live模式。

  3. 用命令行cd到压缩文件存放的目录，一般来说应该是备份在外接设备，比如硬盘里。这里要注意一下，外接设备怎么cd过去：先 cd / ，也就是切换到根目录然后 cd media ，然后 cd ubuntu 后就能看到外接设备。

  4. 现在来到放置备份文件的目录里了，开始执行备份： gzip -dc /part.img.gz | sudo dd of=/dev/sda1，黑斜体的是需要根据自己情况修改的部分，part.img.gz 这个是备份压缩文件名，sda1这个是需要覆盖的分区，怎么看呢，用 fdisk –l 查看，我的是这样

看最右侧一列，倒数第二行有个linux，对应的是sda8，所以应该是覆盖这个区域。
