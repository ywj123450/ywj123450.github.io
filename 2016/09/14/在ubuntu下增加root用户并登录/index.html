在ubuntu下增加root用户并登录

发表于 2016-09-14   |   分类于 技术

  1. 打开终端。

  2. 输入sudo gedit /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf

  3. 在弹出的编辑框里输入：greeter-show-manual-login=true 保存关闭。

  4. 再在中端中输入：sudo passwd root

  5. 输入你想要的密码，关机重启在多出的登录框里输入root 还有你的密码就好了！
